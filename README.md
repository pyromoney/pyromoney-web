# Pyromoney Web

## Installation

Start an API on `localhost:4000` (see https://github.com/pyromoney/pyromoney-api)

Install `create-elm-app` package:

```
npm i -g create-elm-app
```

Run application:

```
elm-app start
```

## Testing

Install `elm-test` binary:

```
npm i -g elm-test
```

And run tests with this command:

```
elm-test
```
